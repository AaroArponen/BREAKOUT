A Breakout clone built with the aid of instructions found on the Mozilla Developer Network site.
https://developer.mozilla.org/en-US/docs/Games/Tutorials/2D_Breakout_game_pure_JavaScript

MySQL, PHP, HTML-5/JavaScript were used to further develope the game and make the experience dynamic.

The game has been tested on following browsers (powered by Windows 10):
Microsoft Edge
Mozilla Firefox
Internet Explorer
Pale Moon
Google Chrome
Opera

and Safari using a MacBook Air

and Chrome using an Android smartphone
